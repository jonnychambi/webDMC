# DMC Landing (versión simple para GitHub Pages)

Esta versión es **estática** (HTML + CSS) para publicar directo en GitHub Pages, sin build ni dependencias.

## Publicar en GitHub Pages

1. Sube estos archivos a tu repo (`index.html`, `styles.css`, `public/.nojekyll`).
2. Ve a **Settings > Pages**.
3. En **Source**, elige **Deploy from a branch**.
4. Selecciona rama `main` y carpeta `/ (root)`.
5. Guarda y espera la URL pública.

## Edición rápida

- Contenido principal: `index.html`
- Estilos: `styles.css`
